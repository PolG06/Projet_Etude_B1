Ce projet a été réalisé en binôme dans le cadre de notre 1ère année de Bachelor.
Il avait pour objectif d’illustrer notre oral et d’accompagner un rapport structuré de 28 pages portant sur le sujet :
Comment réduire les inégalités d'accès à Internet entre pays développés et pays en développement ?
Nous avons conçu une page web interactive permettant de simuler l’effet d’investissements dans différentes infrastructures numériques.

Fonctionnalités:

  -Interface interactive permettant d’ajuster différentes variables (financements, infrastructures…)

  -Simulation de l’évolution de l’accès à Internet en fonction des investissements

  -Mise à jour dynamique des données via un back-end

Répartition des rôles:

  -Front-end : mon camarade

  -Back-end : moi-même (gestion de la logique, manipulation des données, interactions JS)
